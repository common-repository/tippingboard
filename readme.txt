Create and run your own FREE tipping competitions on horses, greyhounds and trots. 

== Description ==

Create a competition via the mobile app. A shortcode will be emailed to you which you can put on any post or page.

Live real time racing fun. Promote your business via a competition by entering your website and a heading.

View competition on a big screen while it updates in real time at www.tippingboard.com
